# Carrim Chartered Accountants

A luxury boutique accounting firm website built with Next.js 14, React, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

1. Push this project to a Git repository (GitHub, GitLab, or Bitbucket).
2. Import the repository at [vercel.com/new](https://vercel.com/new).
3. Vercel will auto-detect Next.js — click **Deploy**.

No environment variables are required for the current setup.

## Customisation

Update contact details in `src/lib/constants.ts`:

- Email address
- Phone number
- WhatsApp number
- Office address

Replace the placeholder founder image in `src/components/Hero.tsx` with your own professional photo.

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**
